L2スイッチとL3スイッチの違い

  • #L2スイッチ
  • #L3スイッチ
  • #ネットワーク
  • #OSIモデル
  • #ITパスポート

ネットワーク機器のカタログや構成図で見かける「L2スイッチ」と「L3スイッチ」。
見た目はどちらも「LANケーブルを挿す穴がたくさんある箱」ですが、中身の役割は全く異なります。

1行要約
L2スイッチは「同じネットワーク内（MAC）」でデータを配り、L3スイッチは「異なるネットワーク間（IP）」でデータを中継する。

1. 「L」の意味とOSI参照モデル

「L」はLayer（層）の略です。OSI参照モデルのどの階層のデータを使って行き先を決めているかを表します。

L4 トランスポート
L3 ネットワーク IPアドレス
（宛先：住所）
L2 データリンク MACアドレス
（宛先：隣の機器）
L1 物理

2. 役割の違いを図解

「何を見て」「どこへ」送るかが異なります。

3. ネットワーク構成での使い分け

  • ルーター： インターネット（WAN）との接続点。
  • L3スイッチ： 社内の部署間（VLAN間）など、高速なルーティングが必要な場所。
  • L2スイッチ： 各デスクのPCやプリンタをまとめる場所。

4. 比較まとめ

試験対策として、ルーターとの違いも含めて整理します。

項目 L2スイッチ L3スイッチ （参考）ルーター
参照情報 MACアドレス IPアドレス IPアドレス
役割 パケット転送 ルーティング ルーティング ＋ α
主な用途 島ハブ、フロア 社内幹線、VLAN間 インターネット接続
OSI階層 データリンク層 ネットワーク層 ネットワーク層

まとめ

スイッチの種類で迷ったら、以下のキーワードで判断しましょう。

  • L2スイッチ： 「MACアドレス」「同一ネットワーク内」
  • L3スイッチ： 「IPアドレス」「異なるネットワーク間」「ルーティング」

「L3スイッチ＝ルーター機能がついた、めっちゃ速いスイッチ」というイメージでOKです。
