155を2進数に変換する方法

  • #2進数
  • #基数変換
  • #計算問題
  • #MSB
  • #LSB

ITパスポートや基本情報技術者試験で頻出の「基数変換」。
今回は例題として、10進数の「155」を2進数に変換する手順を解説します。筆算の書き方さえ覚えれば、どんな数字でも同じ手順で解けます。

1行要約
商が0になるまで「2」で割り続け、出てきた余りを「下から上へ（逆順に）」並べれば完成する。

1. 割り算（すだれ算）の手順

155を2で割り算し、商と余りをメモしていきます。

2 ) 155 1
2 ) 77 1
2 ) 38 0
2 ) 19 1
2 ) 9 1
2 ) 4 0
2 ) 2 0
1 1 (最後の商)

最後の商「1」から始めて、余りを下から順に拾っていきます。
並べると、10011011 となります。

答え： 10011011₂

2. 検算（重みの足し算）

本当に合っているか、各ビットの重みを足して確認してみましょう。
「1」が立っている場所の重みだけを合計します。

1MSB128
064
032
116
18
04
12
1LSB1
128 + 16 + 8 + 2 + 1 = 155

合計が155になったので、計算は正しいことが証明されました。

3. 重要用語：MSBとLSB

ビット列の両端には名前がついています。試験で問われることがあるので覚えておきましょう。

  • MSB (Most Significant Bit)： 最上位ビット。一番左の桁。最も大きな重み（この場合は128）を持つ。
  • LSB (Least Significant Bit)： 最下位ビット。一番右の桁。最も小さな重み（1）を持つ。

まとめ

  • 変換方法： 2で割り続けて、余りを「下から上へ」読む。
  • 検算方法： 128, 64, 32...の重みを足し算する。
  • MSB/LSB： 左がMost（大）、右がLeast（小）。
