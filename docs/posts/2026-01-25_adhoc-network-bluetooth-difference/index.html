アドホックネットワークとBluetooth

  • #通信技術

「アドホックネットワーク」と「Bluetooth」。
この2つは試験でよくセットで出題されますが、全く別のものを指しています。

1行要約
アドホックネットワークは「通信の形（構成）」のことであり、Bluetoothはそれを実現するための「無線技術（道具）」のこと。

1. 通信の「形」の違い

Wi-Fiなどの無線通信には、大きく分けて2つのつなぎ方（モード）があります。

アドホックネットワークの特徴

  • 「Ad hoc」の意味：「その場限りの」「臨時の」。
  • 特徴：基地局やアクセスポイントが不要。端末さえあれば、その場ですぐにネットワークを作れる。
  • 用途：災害時の通信、センサーネットワーク、AirDropなど。

2. Bluetoothとは（技術）

Bluetoothは、デジタル機器用の近距離無線通信規格の「技術名」です。
この技術を使うと、簡単にアドホックな（1対1や1対多の）接続が作れます。

3. 違いと比較まとめ

試験対策として、この2つの関係を整理します。

項目 アドホックネットワーク Bluetooth
分類 ネットワークの形態（形） 無線通信の技術（規格）
定義 中継局なしで端末同士がつながる仕組み 近距離・低電力の無線規格
関係性 「考え方」や「構成」 それを実現する「手段」の1つ

4. 試験での即判断テクニック

ITパスポート試験で問題文に以下のキーワードが出たら、即座に反応しましょう。

  • 「アクセスポイントを経由せず、端末同士が直接通信」
    → 答え：アドホックネットワーク
  • 「近距離」「低消費電力」「周辺機器の接続」
    → 答え：Bluetooth

まとめ

  • アドホック：「どうつなぐか」（形）。基地局不要のバケツリレー。
  • Bluetooth：「何でつなぐか」（技術）。近距離無線の定番。

Bluetoothを使ってアドホックネットワークを作ることはできますが、アドホックネットワーク＝Bluetoothではありません（Wi-Fiでも作れます）。この包含関係を理解しておきましょう。
