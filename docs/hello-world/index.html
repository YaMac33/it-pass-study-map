Hello World: ブログを開設しました

2024年1月10日 #Misc

これは Minimal Blog の最初の記事です。 このブログは画像の読み込みを排除し、テキストコンテンツを最速で届けることを目的として設計されています。

なぜ作ったのか

最近のウェブサイトは重くなりすぎています。 必要な情報にたどり着くまでに時間がかかりすぎることにストレスを感じていました。 そこで、原点回帰として「HTMLとCSSだけで構成された、読みやすさ重視のブログ」を作ることにしました。

特徴

  • 画像を使用しない（HTTPリクエストの削減）
  • 外部フレームワークへの依存を最小限に
  • ディレクトリ構造によるシンプルな管理

コードの書き方サンプル

技術ブログなので、コードブロックの表示も重要です。 pre タグと code タグを使って以下のように表示されます。 

// JavaScriptのサンプルコード
function hello(name) {
    console.log(`Hello, ${name}!`);
}

hello('World');
シンプルであることは、究極の洗練である。 - レオナルド・ダ・ヴィンチ

これから不定期に技術的な知見やメモを更新していきます。
