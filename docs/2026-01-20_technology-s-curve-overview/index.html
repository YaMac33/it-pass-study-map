技術のSカーブの概要

技術の進化には寿命がある。導入から成熟、そして停滞へ至るプロセスを「S字型」のモデルで解説します。

# 1. 定義（結論）

👉 技術の性能や普及度が「導入期 → 成長期 → 成熟期 → 停滞期」とS字型に推移することを示すモデル

# 2. どのような考え方か

  • 技術の発展を時間の経過（横軸）と性能・価値（縦軸）の関係で捉えたもの。
  • 最初はゆっくり、途中から急激に伸び、最後は頭打ちになる。この形がアルファベットの「S」に見える。
  • 企業が「いつ新技術に投資すべきか」「いつ撤退すべきか」を判断する材料になる。

# 4. 成長の4段階

1

導入期 (Introduction)

技術が誕生したばかり。性能向上はまだ小さく、研究開発コストが高い時期。

2

成長期 (Growth)

技術的なブレークスルーにより急速に進歩する時期。性能・効率が大きく向上し、市場が拡大する。

3

成熟期 (Maturity)

技術が安定し、普及する。改良による性能向上は頭打ちになり、コスト削減や効率化が中心となる。

4

停滞期・衰退期 (Decline)

技術進歩が限界に達する。代替となる新しい技術が登場し、主役の座を譲る時期。

# 5. ITパスポート試験でのポイント

  • ！ 「技術の限界」という言葉がキーワード。
  • ！ 成熟期以降は、既存技術に固執せず「次世代技術（新しいSカーブ）」への移行が必要。
  • ！ この移行に失敗すると「イノベーションのジレンマ」に陥る可能性がある。
❌ よくある間違い

「成熟期になっても、投資さえ増やせば性能は無限に伸び続ける」

⭕ 正解

「どんな技術にも物理的・理論的な限界があり、いずれ頭打ちになる」

# 6. 比較：Sカーブ vs 技術ポートフォリオ

観点 技術のSカーブ 技術ポートフォリオ
見る方向 時系列（過去〜未来） 同時点の比較（今どう配分するか）
主目的 技術の寿命・限界を知る 研究開発リソースの配分
判断内容 「いつ技術を乗り換えるか」 「どの技術に重点投資するか」

まとめ（一言）

「技術にも寿命がある。S字を描いて、いつかは止まる。」
